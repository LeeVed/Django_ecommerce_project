# Приложение Catalog




# Установка

Создайте новую директорию для проекта и настройте виртуальное окружение:

    poetry install

Установите при помощи poetry Django:

    poetry add django

Инициализируйте новый проект Django внутри директории: 

    django-admin startproject config .

Создайте новое приложение под названием catalog:

    python manage.py startapp catalog

Зарегистрируйте новое приложение в settings_sample.py

Настройте маршрутизацию для нового приложения, добавив соответствующие URL-адреса.

два HTML-шаблона: для домашней страницы и для страницы с контактной информацией.
Для стилизации страниц используйте Bootstrap.

Создайте контроллер для отображения домашней страницы и страницы с контактами.

Настройте маршрутизацию для этих контроллеров.

Подключите СУБД PostgreSQL для работы в проекте.

Внесите изменения в настройки подключения в файле settings.py(DATABASES перенастроить, 
все чувствительные данные убрать из настроек и перенести в папку .env, создав еще .env.sample для других пользователей
и разработчиков)

В приложении создайте модели Product и Category и опишите для них базовые настройки.

Создайте миграции для новых моделей и примените миграции.

Создайте суперпользователя и зарегистрируйте модели Product и Category в админке.

Настройте отображение для моделей:
для Category выведите id и name в списке;
для Product выведите id , name , price и category в списке.

Настройте фильтрацию продуктов по категории.

Настройте поиск по полям name и description.

Через инструмент shell заполните список категорий:
    
    python manage.py shell

Сформируйте фикстуры для моделей Category и Product .

Реализовать в связке с инструментом работы с фикстурами кастомную команду для добавления тестовых продуктов.
В команде удалите все существующие данные из базы перед добавлением новых продуктов.

Добавьте новый контроллер product_detail (в views.py) и шаблон product_detail.html в templates/catalog для 
отображения страницы с подробной информацией о товаре. На этой странице показана вся информация о товаре.

Добавьте в шаблон главной страницы home.html код для отображения списка товаров с помощью цикла. 
Чтобы карточки товаров выглядели одинаково, обрежьте отображаемое описание до первых 100 символов 
при помощи фильтра |truncatechars.

Создайте базовый шаблон base.html в папке templates (включает шапку, подвал и стили). 

Создайте подшаблон по новигации для главного меню, который можно будет включать в другие шаблоны.
